# Arcane Solarium

## Descrizione
Arcane Solarium è un RPG testuale sviluppato in Java, basato su una struttura narrativa a più atti. Il gioco implementa un sistema di progressione "data-driven", in cui le statistiche e la configurazione dei personaggi per ogni atto sono gestite tramite file esterni in formato JSON, garantendo modularità e facilità di bilanciamento.

## Caratteristiche Principali
* **Architettura Data-Driven**: Utilizzo di file JSON per la configurazione delle statistiche.
* **Persistenza**: Sistema di salvataggio/caricamento dello stato del giocatore tramite serializzazione.
* **Design Pattern**: Rispetto dei principi SOLID per garantire un codice estensibile e manutenibile.
* **Interfaccia**: Interazione via CLI (Command Line Interface) con gestione dinamica dei flussi di gioco.

## Requisiti
* Java Development Kit (JDK) 17 o superiore
* Gradle
